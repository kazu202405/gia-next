# 行動科学 × ビジネスアイデア

行動心理学・脳科学・行動経済学の知見を核にしたビジネス案。

## 1. 行動設計コンサルティング（BtoB）

**一言**: 企業のあらゆる「人が動かない問題」を行動科学で解決する

### コンセプト
マーケティング会社でもデザイン会社でもない。「人間の行動を変える専門家集団」というポジション。
クライアントの課題を認知バイアス・脳科学の観点で診断し、科学的根拠のある介入を設計する。

### なぜ効くか（心理学的根拠）
- 多くの企業は「なぜユーザーが動かないか」を感覚で判断している
- 行動科学のフレームワーク（BJ Fogg、ナッジ理論）で構造化すると、打ち手が明確になる
- 「勘と経験」→「科学と検証」への転換は説得力が高い（権威の原理）

### サービスライン
| サービス | 内容 | 単価感 |
|---------|------|--------|
| 行動診断 | サイト/アプリ/サービスの心理学的監査 | 
| CRO設計 | コンバージョンの心理学的最適化 |
| 価格設計 | アンカリング・おとり効果を活用した料金体系 | 
| オンボーディング設計 | 初期体験の心理学的設計 | 
| 社内行動変容 | 従業員の行動を変えるナッジプログラム | 
| 研修・ワークショップ | 行動科学リテラシーの社内浸透 | 

### ターゲット
- SaaS企業（解約率・アクティベーション率に悩む）
- EC企業（カート放棄率、リピート率に悩む）
- 人事部門（従業員エンゲージメント、研修効果に悩む）


### 競争優位
- 「行動科学専門」という明確なポジショニング（日本にほぼ不在）
- すべての提案に論文・研究の裏付けがある（権威の原理で受注率UP）
- 効果測定まで含めた科学的アプローチ

## 3. 行動科学メディア / コミュニティ（情報ビジネス）

**一言**: 「人間の取扱説明書」を発信し、実務家コミュニティを形成する

### コンセプト
行動心理学・脳科学・行動経済学の知見を「日常 × ビジネス」の両面で発信するメディア。難しい論文を「明日使える形」に翻訳する。コミュニティで実務家同士の学びを加速。

### コンテンツ軸
| カテゴリ | 例 |
|---------|-----|
| 日常の心理学 | 「先延ばしの脳科学」「なぜパートナーと喧嘩するのか」 |
| ビジネス応用 | 「料金ページの心理学」「1通のメールでCVR2倍にした方法」 |
| 論文解説 | 最新の行動科学研究を5分で読める形に |
| ケーススタディ | 企業の行動設計事例の分析 |
| 実験レポート | 自分たちで試したA/Bテストの結果共有 |
```

### なぜこれが強いか
- **好奇心ギャップ**: 「なぜ人は○○するのか」というテーマは本質的にクリックされやすい
- **実用性**: マーケター・デザイナー・マネージャー全員が使える
- **日常 × ビジネスの二面性**: 間口が広く、ファン化しやすい
- **コミュニティ効果**: 社会的証明 + 所属感 + 学びの返報性で継続率が高い

## 4. 意思決定デバイアス・ツール（BtoB SaaS）

**一言**: 経営判断・採用・投資の「認知バイアス」をリアルタイムで可視化する

### コンセプト
重要な意思決定の場面（経営会議、採用面接、投資判断）で、チームが陥っている認知バイアスを検出し、より良い判断を支援するツール。

### 仕組み
1. 会議の議事録・意思決定プロセスをインプット
2. AIが認知バイアスのパターンを検出
3. 「確証バイアスの兆候があります」「アンカリングに引きずられている可能性」などリアルタイム表示
4. デバイアスのための具体的な問いを提示（「逆の視点から見ると？」「10倍の規模だったら同じ判断をする？」）

### 適用場面
| 場面 | 頻出バイアス | 介入方法 |
|------|------------|---------|
| 経営戦略 | 確証バイアス、サンクコスト | プレモーテム、悪魔の代弁者 |
| 採用面接 | ハロー効果、類似性バイアス | 構造化面接、ブラインド評価 |
| 投資判断 | アンカリング、自信過剰 | ベースレート提示、レファレンスクラス |
| プロダクト判断 | IKEA効果、現状維持バイアス | ユーザーデータ強制参照 |

### ターゲット
- 経営コンサルティングファーム
- VC / PE（投資判断の質向上）
- 大手企業の経営企画部門
- 採用に課題を持つ成長企業

##  説得・交渉トレーニング（BtoB / BtoC）

**一言**: 営業・交渉・プレゼンを「才能」から「科学」に変える

### コンセプト
チャルディーニの説得原理、フレーミング、アンカリングなどを実践的に訓練するプログラム。座学ではなくシミュレーション重視。

### プログラム構成
```
Level 1: 基礎理論（オンライン / 2時間×4回）
├── 人間の意思決定メカニズム（System 1/2）
├── 6つの説得原理の理解と日常観察
├── 認知バイアスの体系的理解
└── 自分のバイアスに気づくワーク

Level 2: 実践訓練（ワークショップ / 1日×2回）
├── 営業シナリオでのロールプレイ
├── 価格交渉シミュレーション
├── プレゼンのフレーミング設計
└── 反論対処の心理学的アプローチ

Level 3: 応用・定着（コーチング / 月2回×3ヶ月）
├── 実案件への適用サポート
├── 録画レビューとフィードバック
├── 高度な交渉戦略の設計
└── チーム全体への浸透支援
```

### ターゲット
- 営業チームを持つBtoB企業
- 不動産・保険・金融の営業組織
- スタートアップの創業者（資金調達プレゼン）
- フリーランス・コンサルタント（単価交渉）

### 行動科学の組み込み方
- **Tiny Habits（BJ Fogg）**: 「腕立て2回」から始める超小型習慣
- **習慣スタッキング**: 既存の習慣（歯磨き後）に新習慣をくっつける
- **環境設計**: 物理環境の変更を具体的に提案
- **実行意図**: 「いつ・どこで・何を」を事前設定
- **変動報酬**: 予測不能な小さなご褒美で飽きさせない
- **社会的契約**: 友人に宣言する仕組み（コミットメント効果）
- **アイデンティティ変容**: 「ランニングする人」→「ランナーである自分」への移行支援


| 機能 | 行動科学的根拠 |
|------|--------------|
| 困った行動の診断 | 行動のABC分析（先行条件→行動→結果）|
| 環境設計の提案 | ナッジ理論（片付けやすい収納配置など） |
| 声かけ変換 | 「○○しなさい」→「AとBどっちにする？」（選択肢の提示） |
| ほめ方ガイド | 成長マインドセット（結果より努力・プロセスをほめる） |
| 年齢別行動ガイド | 脳の発達段階に応じた期待値の調整 |
| 親のストレス管理 | 認知的再評価、6秒ルール |

### 低リスク・早期収益（まずはここから）
| 順位 | ビジネス | 理由 |
|------|---------|------|
| 1 | **メディア/コミュニティ（案3）** | 初期投資ほぼゼロ。発信で権威性を構築し、他すべてのビジネスの集客基盤になる |

## この事業の本質的な強み

**「人間は変わらない」ということ。**

テクノロジーは変わる。トレンドも変わる。でも人間の脳の構造、認知バイアス、感情のメカニズムは何万年も変わっていない。

- アンカリング効果は1974年にカーネマンが発見して、2026年の今も同じように機能する
- 損失回避は原始時代の生存戦略の名残で、AIの時代でも変わらない
- 社会的証明は人類が集団生活を始めた時から今日のSNSまで同じ原理

だから行動科学のスキルは**陳腐化しない**。プラットフォームが変わっても、ツールが変わっても、「人間の取り扱い方」を知っている人は常に価値がある。

##  経営の解像度を上げる行動科学パートナー

**一言**: 経営者の「判断のOS」をアップデートし、見えていなかった盲点を可視化する継続伴走サービス

### コンセプト

従来コンサルが「何をすべきか（What）」を教えるのに対し、本サービスは「どう判断すべきか（How to decide）」を扱う。経営課題は変わっても意思決定のOSは共通であるため、一度改善すれば全ての判断に波及する。

| 項目 | 従来型コンサル | 行動科学パートナー |
|------|--------------|------------------|
| 提供価値 | 「正解」を教える | 「認知の歪み」に気づかせる |
| 前提 | 経営者は合理的に判断できる | 経営者もバイアスの影響下にある |
| 関与方法 | 短期プロジェクト型 | 継続的伴走型 |
| 対象 | 事業課題（市場、戦略、財務） | 意思決定プロセスそのもの |
| 依存関係 | コンサルへの依存を生みやすい | 自走力（メタ認知）の獲得が目標 |

### ポジショニング

```
                    短期              長期
                ┌──────────────┬──────────────┐
  戦略コンサル    │  McKinsey    │              │
  (What to do)  │  BCG, Bain   │              │
                ├──────────────┼──────────────┤
  行動科学       │  セミナー     │ ★本サービス  │
  (How to decide)│  研修会社    │  パートナー   │
                └──────────────┴──────────────┘
```

「How to decide × 長期伴走」の象限には実質的な競合がほぼ不在。

### フック：経営バイアス・プロファイリング診断シート
これがページか遷移かモーダルかであってもいいかも？

#### 設計思想

回答すること自体が自己発見のプロセスになるよう設計。

- **好奇心ギャップ**（Loewenstein, 1994）：「自分が知らない自分」を示唆し続きを知りたくさせる
- **IKEA効果**：自分で回答に関与したことで結果への投資感が生まれる
- **ツァイガルニク効果**：診断結果の一部だけを見せることで「残り」が気になる
- **バーナム効果の逆利用**：回答パターンごとに異なる具体的フィードバックを返すことで信頼性を担保

#### 構成：6領域 × 各3問 ＝ 全18問（所要時間：約7分）

各問は5段階リッカート尺度（1：全くそうでない〜5：非常にそうである）

---

**領域A：意思決定の慣性（→ Module 1 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| A-1 | 重要な経営判断をするとき、「前回うまくいった方法」を最初に検討することが多い | 現状維持バイアス、アンカリング |
| A-2 | 事業の撤退や方針転換の判断が、始めた当初より難しくなっていると感じることがある | サンクコスト効果 |
| A-3 | 新しい戦略を検討するとき、まず「今のやり方の延長線上」で考えることが多い | 現状維持バイアス、フレーミング |

**領域B：情報の取り扱い（→ Module 2 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| B-1 | 自分の仮説を裏付けるデータが見つかると、安心してそれ以上の検証を省略することがある | 確証バイアス |
| B-2 | 信頼している人物や専門家の意見は、データや数字よりも判断に影響することがある | 権威バイアス、ハロー効果 |
| B-3 | 最近起きた出来事（業界ニュース、競合の動きなど）が、中長期の戦略判断に過度に影響していると感じることがある | 利用可能性ヒューリスティック |

**領域C：リスクと損失の感覚（→ Module 3 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| C-1 | 「今ある利益を守る」判断と「新たな利益を取りに行く」判断では、前者を選ぶことが多い | 損失回避（プロスペクト理論） |
| C-2 | 一度投資した事業やプロジェクトに対して、「ここまでやったのだから」という理由で追加投資を決めたことがある | サンクコスト錯誤 |
| C-3 | 成功確率70%の案件と失敗確率30%の案件を、同じ案件だと頭では分かっていても異なる印象を受ける | フレーミング効果 |

**領域D：組織・人材の認知（→ Module 4 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| D-1 | 部下やチームメンバーが成果を出せなかったとき、まず「その人の能力や姿勢」に原因を求めることが多い | 基本的帰属錯誤 |
| D-2 | 「うちの社員はもっと自主的に動くべきだ」と感じることがある | 自己決定理論の誤解 |
| D-3 | 組織の問題について、「仕組み」よりも「人」を変えることで解決しようとすることが多い | 環境設計vs意志力 |

**領域E：時間軸の歪み（→ Module 5 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| E-1 | 短期的な成果（今月の売上、四半期の数字）が、長期的な投資判断に影響することがある | 双曲割引、現在バイアス |
| E-2 | 「今すぐ着手する小さな改善」と「半年後に大きな成果を生む施策」では、前者を選びがちである | 双曲割引 |
| E-3 | 過去の成功体験について、「あのときの判断は正しかった、最初からうまくいくと思っていた」と感じることがある | 後知恵バイアス |

**領域F：自己認知の精度（→ Module 6 に対応）**

| # | 設問 | 測定バイアス |
|---|------|-------------|
| F-1 | 自分の判断に対する確信度（「これは絶対にうまくいく」「間違いない」）は高い方だと思う | 過信バイアス、ダニング＝クルーガー |
| F-2 | 会議や議論で、自分の意見が場の空気に影響を与えていることを意識することは少ない | HiPPO効果 |
| F-3 | 自分のリーダーシップスタイルについて、周囲が率直にフィードバックしてくれる環境があると思う | 心理的安全性 |

#### Module 1：意思決定のOS書き換え
**診断領域A対応 | 理論：二重過程理論、アンカリング、現状維持バイアス、サンクコスト**

| ワーク | 内容 | 時間 |
|--------|------|------|
| 意思決定ジャーナル分析 | 直近1ヶ月の重要判断5件をSystem1/2で分類し、無意識の慣性を可視化 | 60分 |

### 導線設計

```
Stage 0：認知・興味
  └─ セミナー/記事/SNS「あなたの判断を歪めている3つのバイアス」
     ↓（好奇心ギャップ）

Stage 1：診断シート（無料・7分）
  └─ オンラインで即時結果 → 一部だけ開示
     ↓（ツァイガルニク効果：詳細を知りたい）

Stage 2：無料30分セッション
  └─ 診断結果の読み解き → 盲点の深掘り
     ↓（ラベリング効果：「あなたは〇〇型の意思決定者です」）

Stage 3：カリキュラム受講（有料・6ヶ月）
  └─ 全6モジュール × 月1回3時間
     ↓（コミットメントと一貫性）

Stage 4：継続パートナーシップ（月額）
  └─ 月1回1on1 + 経営判断レビュー + 四半期再診断
     ↓（習慣化、アイデンティティ変容）

Stage 5：組織展開
  └─ 経営チーム全体 → 管理職 → 意思決定プロトコル設計
```

**各転換ポイントの仕掛け**

| 転換 | 原理 | 仕掛け |
|------|------|--------|
| 認知→診断 | 好奇心ギャップ | 「経営者の93%が自覚していない盲点」 |
| 診断→セッション | ツァイガルニク + 損失回避 | 「詳細はセッションで」+「放置のリスク」 |
| セッション→カリキュラム | コミットメント + 社会的証明 | 小さなYesを引き出す + 同業の変化事例 |
| カリキュラム→パートナー | 保有効果 + 損失回避 | 「せっかく上げた解像度を落とさない」 |
| パートナー→組織展開 | 社会的証明 + 一貫性 | 「あなたが変わったように、チームも」 |
