あなたはシニアUI/UXデザイナー兼Reactフロントエンドエンジニアです。

# 目的
「空間・建築デザイン」事業の詳細ページ（サービス紹介LP）を1ページ制作してください。
単なる施工紹介ではなく、
“空間体験・動線・心理まで設計するハイエンド空間デザインブランド”
として表現することがゴールです。

# デザインコンセプト
素材（写真・イラスト）を最小限に抑えた、コードベースのミニマルデザイン。
高級感と静謐さを表現した日本的モダンWebサイト。
Apple / Lusion / Active Theory のような上質で体験型のUI。

# 技術スタック
- React + Vite + Tailwind CSS 
- GSAP
- Lenis
- SVGのみで装飾生成（画像素材禁止）

# 必須制約
- 外部画像・アイコン禁止
- すべてコード/SVG/CSSのみで表現
- 1ファイルのReactコンポーネントで出力
- レスポンシブ

# 表現スタイル
- ジェネラティブ背景（SVG blob morphing）
- ノイズテクスチャ（feTurbulence）
- 浮遊パーティクル
- セクション波型ディバイダー
- タイポグラフィ中心
- 大きな余白
- 深緑アクセント
- 静かで高級感のある動き（派手禁止）

# ページ構成
1. Hero（大きなタイポ＋ジェネラティブ背景）
2. コンセプト説明
3. 提供価値（心理・動線・体験）
4. サービス内容
5. プロセス
6. 事例（画像なしカード）
7. CTA



# 出力
- React TSXコード
- Tailwind使用
- GSAPアニメーション込み
- コメント付き
